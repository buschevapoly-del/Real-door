Consent, correction, and deletion

  • You confirm or correct every extracted value before it is used in any calculation.
  • Uploaded PDFs are processed in memory and deleted immediately after extraction -- the file itself is never stored.
  • Only the allowlisted field values above are kept, in a single file per household.
  • An activity log records what actions were taken (uploaded, corrected, confirmed, exported, deleted) and when -- never the document contents or the corrected values themselves.
  • Deleting your package removes that file, including its activity log, entirely.
  • Nothing is ever sent automatically to a property or provider -- export is the only way data leaves the app, and only when you choose it.
